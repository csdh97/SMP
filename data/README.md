## Datasets


1. The raw [pseudo-dimer](https://dataverse.harvard.edu/file.xhtml?fileId=11632259&version=1.1) dataset is available for pre-training protein interaction models. This dataset contains structural information for pseudo-dimer complexes.
2. The raw **protein–protein contact map prediction** datasets with structural information can be downloaded from [contact_dataset_homo](https://dataverse.harvard.edu/file.xhtml?fileId=11666621&version=1.1) and [raw_contact_data_hetero](https://dataverse.harvard.edu/file.xhtml?fileId=11666624&version=1.1). In addition, the test set features for contact prediction are available at [contact_test_set](https://huggingface.co/datasets/dh97/contact_test_set), which can be used to reproduce the results reported in our manuscript.
3. The raw **protein–protein docking** dataset with structural information can be accessed via [raw_docking_dataset](https://dataverse.harvard.edu/file.xhtml?fileId=11674560&version=1.1). While the processed dataset can be download through[processed_docking_dataset](https://dataverse.harvard.edu/file.xhtml?fileId=11674560&version=1.1).Samples that could not be parsed by the EquiDock preprocessing pipeline have been removed.
4. The protein–protein interaction datasets can be downloaded from [interaction_dataset_DSCRIPT_train](https://dataverse.harvard.edu/file.xhtml?fileId=11688742&version=1.1), [interaction_dataset_DSCRIPT_val](https://dataverse.harvard.edu/file.xhtml?fileId=11688825&version=1.1), [interaction_dataset_DSCRIPT_test](https://dataverse.harvard.edu/file.xhtml?fileId=11691785&version=1.1), and [interaction_dataset_HIPPIE](https://dataverse.harvard.edu/file.xhtml?fileId=11675120&version=1.1). These include both the D-SCRIPT and HIPPIE datasets.
